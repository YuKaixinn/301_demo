# 心理特质预测模型报告 (Psy Prediction Report)

## 1. 项目摘要

应您的要求，我们构建了专门用于预测 **心理特质 (Psy_score)** 的模型，探究其他多模态指标（体能、生理、动机等）在多大程度上能反映一个人的心理状态。

### 1.1 模型概况
*   **目标**: 区分高心理素质 vs 低心理素质人群 (基于 Psy_score)
*   **数据分组**: **Top 30% vs Bottom 30%** (剔除中间 40%)。此分组策略下区分度最高。
*   **最佳模型**: **Logistic Regression + Polynomial Features** (多项式逻辑回归)
*   **模型精度 (AUC)**: **0.833** (达成 > 0.8 的目标)

### 1.2 降维流程与变量构成
本模型使用与综合成绩预测相同的 PCA 降维特征（Eye, ECG, EMG, Cog, Ele 等），详情请参考综合成绩预测报告中的特征工程部分。

## 2. 关键预测因子 (Key Predictors)
模型发现以下特征是预测心理素质的核心指标：

| 排名 | 特征 (Feature) | 权重/重要性 | 解释 (Interpretation) |
| :--- | :--- | :--- | :--- |
| **1** | **Motivation (动机)** | **极高 (1.13)** | **意愿决定心态**。内部动机越强，心理韧性越强。这是最强的单一预测因子。 |
| **2** | **EMG * Ele (肌电×体能)** | **高 (0.93)** | **身心交互**。优秀的体能配合良好的肌肉控制，往往伴随着更稳定的心理状态（“艺高人胆大”）。 |
| **3** | **EMG (肌电)** | **负相关 (-0.89)** | **肌肉紧张度**。过高的肌肉紧张水平（不必要的用力）通常对应较低的心理放松度/抗压能力。 |
| **4** | **ECG (心电)** | **高 (0.83)** | **生理冷静度**。HRV 越高（心电调节越好），心理弹性评分通常越高。 |

## 3. 结论
*   **心理不是孤立的**: 心理素质与**动机水平**高度绑定，同时深深植根于**生理状态** (心电、肌电) 之中。
*   **身心一体**: 体能好且肌肉控制能力强的人，往往表现出更强大的心理抗压能力。
